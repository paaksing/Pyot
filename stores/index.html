# Legend

-Pyot Cache: Identifies the store of type Cache.

-Pyot Service: Identifies the store of type Service.

-Sharding: This store supports multiple instances (hence sharding by separating multiple Cache to handle different type of objects) in the Pipeline.

-Model: This is a Pyot model.

-<model-name>: The name of the model supported if not all.

TIP

The arguments keys passed to Pyot Settings Pipeline can be in ALL CAPS to maintain better code redability, it is automatically lowercased on activation anyways.
